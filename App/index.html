The MOC Combat system allows players to juggle a series of upgrades and downgrades (Enhanced Attributes) to get exactly the flavor of creation they're looking for. That time is almost always better spent building more weapons instead. The easiest way to calculate a creation's core attributes is not to bother (Default Attributes). For creation stats that don't need personalized attention, players can drop in default abilities with a simple Size measurement like you want it. (MC.1.2 Default Attributes)

The base Unit Inch Value of a unit (minifigs, autonomous laser turrets, artificially intelligent spider robots) is equal to its inches of Size. In MOC Combat, minifigs and non-Heroic minifig Specialists are each worth one Unit Inch. Minifig Heroes are worth two Unit Inches. See table of MC.4.2 Measuring Units for more information.

A basic object in BrikWars starts with Armor 1d6 and no movement ability. If it's able to operate devices, it has enough Power to operate twice its own Size inches' worth of weapons. If it has the ability to act on its own, it does so with an Action die of 1d6.
As a creation's Size increases, it gains Enhancements to these basic attributes, described in detail in their respective chapters. Creatures and vehicles gain one Enhancement for each of their first five inches of Size. Unmoving structures gain Enhancements on odd-numbered Size inches (1", 3", and 5").
For either type, additional Size inches past 5" have no further benefit. Any Enhancement added past these limits must be paid for with an Impairment.
